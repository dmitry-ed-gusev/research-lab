Пакет содержит классы, реализующие конфигурирование системы массовой рассылки почты. BaseMailerConfig - базовый класс с основными параметрами, необходим для разделения функциональности между классами (упрощение классов), данным классом напрямую не следует пользоваться в данной системе. MailerConfig - расширяет базовый класс конфигурации несколькими полями и возможностью разбора параметров командной строки (массив String args[]).
