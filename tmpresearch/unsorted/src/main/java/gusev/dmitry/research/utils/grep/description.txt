Задача: реализовать консольную утилиту аналог grep.
=====================================================================
Необходимо реализовать консольную программу, которая бы фильтровала поток текстовой информации подаваемой на
вход и на выходе показывала лишь те строчки, которые содержат слово передаваемое программе на вход в качестве
аргумента.

Варианты усложнения:
---------------------------------------------------------------------
1. Программа не должна учитывать регистр
2. В аргументах может быть передано не одно слово, а несколько
3. В качестве аргумента может быть задано не конкретное слово, а регулярное выражение
=====================================================================

15.10.2012 Детали реализации:
- программа консольная
- с ключом командной строки -s указывается слово/слова для поиска (одно за другим, через пробел)
- с ключом командной строки -r программа игнорирует регистр символов
- с ключом командной строки -f можно указать входной текстовый файл, в котором будет осуществляться поиск.
  Если данный ключ не указан, то программа будет считывать строки из консоли до появления команды (строки)
  вида [go!]
