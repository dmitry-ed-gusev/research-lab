В пакете находятся модули для пакетной обработки списков sql-запросов (sql-batch). Запросы в списке обязательно должны быть не-SELECT - только обновление/изменение данных или изменение структуры таблиц/баз данных. Перед выполнением списки запросов фильтруются (pre processing) и из них удаляются пустые записи и запросы типа SELECT. Модули пакетной обработки однопотоковые и многопотоковые, многопотоковые модули следует использовать с осторожностью.
