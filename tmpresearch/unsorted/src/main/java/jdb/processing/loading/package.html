Данный пакет содержит модуль сериализации/десериализации БД и таблиц поддерживаемых СУБД. При сериализации на диске создается структура каталогов подобная структуре БД и в каждом каталоге находится одна из сериализованных таблиц БД. Файлы, в которые записаны сериализованные данные архивируются с помощью zip-алгоритма.
