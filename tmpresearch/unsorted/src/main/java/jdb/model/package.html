В данном пакете расположены различные модели баз данных - модель структуры БД, модель БД с указанием времени обновления каждой таблицы, модель БД со всеми данными. Эти модели могут использоваться для хранения информации о любой СУБД и для обмена этой информацией между различными процессами - напр., через сериализацию моделей. Также эти модели используются модулем Spider для возвращения пользователю полученной информации о СУБД и из СУБД. Имена всех таблиц и полей хранятся только в ВЕРХНЕМ регистре символов - т.к. в различных СУБД и БД имена могут храниться в разном регистре и фактически одна и та же таблица в одном месте может иметь имя [ИМЯ], а в другом месте - [имя] и эти таблицы будут приняты алгоритмами за разные.
