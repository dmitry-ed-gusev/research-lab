DataTex Connector utility, version 0.1.0, 2018 (C) Larga

Утилита-коннектор между системами:
  - DataTex DBMS
  - LuxMS  BI
  - BFG

Утилита представляет собой консольное приложение с набором ключей. Конфигурирование утилиты осуществляется с
помощью файла [environment.yml] в корневом каталоге утилиты, параметры конфигурации очевидны и прокомментированы в
самОм файле.

Запуск утилиты осуществляется с помощью команды:

    java -jar <utility-name>.jar [--help | [--listds/--listtables]]
